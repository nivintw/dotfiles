One command to a fully provisioned Mac.

A declarative, idempotent dotfiles repo. home/ mirrors $HOME and is symlinked into place with GNU Stow; everything else — tools, shell, editor, security posture — is reproduced from a single bootstrap script.

#Principles

The constraints every change in this repo is held to.

01

Idempotent by default

install.sh is safe to re-run: it adopts already-present state instead of clobbering it, and never re-does work it's already done.

02

One source of truth

Editing ~/.config/fish/config.fish edits the tracked file directly — Stow makes the dotfile in $HOME and the repo the same file.

03

Secrets never committed

Credentials are 1Password op:// references resolved at install time. gitleaks and a custom SSH-host guard fail the commit if anything concrete slips in.

04

Proportionate testing

Cheap static analysis on every commit, unit tests for the one piece of real logic, and a manual VM checklist for the bootstrap. No test heavier than the thing it tests.

#Quick start

No prerequisites — the script installs Homebrew and uv itself if they're missing.

git clone <this-repo> ~/dotfiles
~/dotfiles/install.sh
Idempotent
Re-run it any time. brew bundle adopts already-installed casks in place, stow refuses to clobber real files, and every step skips work it has already done.
