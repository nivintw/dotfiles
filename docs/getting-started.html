Getting Started

No prerequisites — install.sh installs Homebrew and uv itself if they're missing. Clone, run, done.

git clone <this-repo> ~/dotfiles
~/dotfiles/install.sh
Idempotent
Safe to re-run at any point. brew bundle adopts already-installed casks in place, stow refuses to clobber existing real files, and MCP/symlink/shell steps all no-op when already done.

#What the bootstrap does

In order — privileged steps are grouped into a single sudo session so you authenticate once.

#StepWhat happens
0ToolchainInstall Homebrew & uv if missing
1brew bundleAll CLI tools, fish, GUI casks, the MesloLGS NF font
2Privileged blockTouch ID for sudo (sudo_local + pam_reattach), fish as login shell, app firewall + stealth mode — one sudo prompt
3StowSymlink home/ into $HOME
4SSHCreate empty ~/.ssh/config.local for machine-local hosts
5fisherfish plugins from fish_plugins (tide, fzf.fish)
6TPMtmux plugin manager + plugins from tmux.conf
7atuinBackfill existing shell history into atuin's DB
8iTerm2Point iTerm at the tracked iterm2/ prefs folder
9uv toolsInstall the Python CLIs in uv_tools.txt
10git templateprek hook shims in ~/.config/git/template (auto-install on clone)
11Claude CodeNative installer (self-updating) → ~/.local/bin/claude
12MCP serversReplay claude_mcp.json into ~/.claude.json (secrets via op inject)
13OllamaStart the server & pull qwen2.5-coder:7b for GitLens' local AI
14macOS defaultsmacos.sh system tweaks
15DockDeclarative Dock layout via dock.sh
iTerm2
The iTerm step takes effect on iTerm's next launch — fully quit it first if it's already open.

#The human-only steps

A few things no script can do for you.

›

Claude Code login

The CLI is installed automatically; first run needs a one-time browser sign-in (claude). MCP servers are already registered.

›

GitHub MCP token

Store a PAT in 1Password at op://MCP/github-claude-pat/credential, enable the 1Password CLI integration, then re-run the script — op inject bakes it into ~/.claude.json, never the repo.

›

VS Code & 1Password

Sign in to VS Code to sync, and add the 1Password browser extensions (both apps are casks).

›

AppCleaner SmartDelete

A one-time GUI toggle (installs a privileged helper) — can't be scripted via defaults.

#Updating

+

New CLI tool / app

Add a brew/cask line to the Brewfile, then brew bundle install --file=~/dotfiles/Brewfile.

+

Dotfile change

Edit the file under home/ (or its symlink in $HOME — same file) and commit.

+

Find untracked installs

brew bundle cleanup --file=~/dotfiles/Brewfile reports prune candidates.
